# 💪 Gym Schedule Manager

Gym Schedule Manager adalah aplikasi web sederhana untuk mengelola jadwal latihan gym.
Aplikasi ini dibuat menggunakan **Python (Flask)** sebagai backend dan **JavaScript + LocalStorage** untuk penyimpanan data.

---

# Fitur

* Menambahkan jadwal latihan gym
* Menampilkan daftar jadwal latihan
* Mengedit jadwal latihan
* Menghapus jadwal latihan
* Penyimpanan data menggunakan LocalStorage

---

# Teknologi yang Digunakan

* HTML
* CSS

# Cara Menjalankan

### 1. Clone repository

```
git clone https://github.com/username/gym-schedule-manager.git
```

### 2. Masuk ke folder project

```
cd gym-schedule-manager
```

### 3. Install Flask

```
pip install flask
```

### 4. Jalankan aplikasi

```
python app.py
```

### 5. Buka di browser

```
http://127.0.0.1:5000
```
